SVN

Posted on 2018-05-04 | In 版本管理

Xcode默认自带Git 与 SVN，

######首先检测你的Mac 中是否已经安装了SVN，Mac os 10.8的用户没有的。
检测的方式如下：
打开终端，输入以下命令

svn -version

此时，如果已经安装了svn，那么终端控制台将出现如下信息：

svn: invalid option character: e
Type 'svn help' for usage.

如果出现如下，则说明当前系统没有安装svn

svn: command not found

如果还没有SVN，解决方法如下：

  1. 进入Xcode -> “Preferences”快捷键：Command+,）-> “DownLoads”，然后下载安装Command Line Tools

Xcode -> “preferences” -> “Accounts”
在弹出的对话框中Address:输入服务器地址，Type选择Subversion，输入账号密码之后点Add，配置ok了。
如何使用呢？
Xcode -> “Source Control”
Check Out是下载服务器上的代码，Update的时更新，Commit是上传。

扒一扒MAC上好用的SVN管理工具

Posted on 2018-05-04 | In 版本管理

俗话说：“工欲善其事必先利其器”；现在各个公司用于源码管理工具通常有下面两种：

第一种，Git：使用git的公司应该是最多的。

第二种，svn：目前我们公司使用的就是svn。

####Cornerstone(69$)(首推)：

软件首页：http://www.zennaware.com/cornerstone/index.php

下载链接(国内市场不支持)：http://itunes.apple.com/us/app/cornerstone/id404789253?mt=12&ls=1

#####破解方法–无限期试用（先下载试用版本）：

  1. 修改系统时间（如果你从来没有安装过这个试用版）

    安装的时候把系统时间改到超前或者超后的时间就可以，比如2050年或者1900年，安装完成后再把系统时间改回来就可以了。然后一辈子都是14天试用。(亲测可以我把系统时间改称为2030年，我用的3.0.3版本)

  2. 删除试用日期log

    在试用快过期之前，显示系统隐藏文件，然后前往文件：~/Library/Preferences，删除Preferences文件下的ByHost文件夹，这样就会从第一天开始计算试用天数。或者找到ByHost文件夹进去，打开com.zennaware.Cornerstone开头的plist文件，修改plist文件中com.zennaware.Cornerstone:2对应的日期Data，目测可行(未测，谁用过可以告诉我)
